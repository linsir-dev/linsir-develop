# Java 面向对象编程示例

本目录包含 Java 面向对象编程的示例代码，展示了封装、继承、多态、接口、组合等核心概念。

## 目录结构
```
oopdemo/
├── Animal.java         // 动物抽象类
├── Cat.java            // 猫类（继承Animal，实现Walkable）
├── Dog.java            // 狗类（继承Animal，实现Walkable）
├── Main.java           // 主测试类
├── PetOwner.java       // 宠物主人类（组合关系）
└── Walkable.java       // 可行走接口
```

## 示例代码功能

### 1. 抽象类 (`Animal.java`)
- **封装**：
  - 使用 private 字段和 public getter/setter 方法
  - 保护数据的安全性和完整性
- **抽象方法**：
  - 定义了抽象方法 `speak()`，由子类实现
  - 展示了多态的基础
- **具体方法**：
  - 提供了 `eat()` 等通用方法
  - 子类可以直接使用或覆盖

### 2. 接口 (`Walkable.java`)
- **抽象方法**：
  - 定义了 `walk()` 方法
  - 实现类必须提供具体实现
- **默认方法**：
  - 提供了 `stroll()` 默认方法
  - 实现类可以选择覆盖或使用默认实现
- **使用场景**：
  - 定义行为契约
  - 支持多重继承

### 3. 继承与实现 (`Dog.java`, `Cat.java`)
- **继承**：
  - 继承自 `Animal` 抽象类
  - 实现抽象方法 `speak()`
  - 复用父类的代码
- **接口实现**：
  - 实现 `Walkable` 接口
  - 提供 `walk()` 方法的具体实现
- **多态**：
  - 不同子类对同一方法的不同实现

### 4. 组合关系 (`PetOwner.java`)
- **组合**：
  - 宠物主人拥有多个宠物（`List<Animal>`）
  - 展示了 "has-a" 关系
- **多态应用**：
  - 通过 `Animal` 引用调用 `speak()` 方法
  - 运行时根据实际类型执行相应的实现
- **接口使用**：
  - 使用 `instanceof` 检查对象是否实现了 `Walkable` 接口
  - 进行类型转换并调用接口方法

### 5. 测试类 (`Main.java`)
- **完整演示**：
  - 创建宠物主人和宠物对象
  - 演示宠物的收养
  - 展示多态的实际应用
  - 测试接口方法的调用

## 面向对象编程核心概念

### 1. 封装（Encapsulation）
- 将数据和行为封装在类中
- 通过访问修饰符控制访问权限
- 提供公共接口，隐藏实现细节

### 2. 继承（Inheritance）
- 子类继承父类的属性和方法
- 实现代码复用
- 支持方法重写，实现多态

### 3. 多态（Polymorphism）
- 同一类型的对象表现出不同的行为
- 通过方法重写和接口实现
- 提高代码的灵活性和可扩展性

### 4. 抽象（Abstraction）
- 通过抽象类和接口定义行为契约
- 隐藏具体实现细节
- 关注 "做什么" 而不是 "怎么做"

### 5. 接口（Interface）
- 定义行为规范
- 支持多重继承
- 实现松耦合设计

### 6. 组合（Composition）
- "has-a" 关系
- 一个对象包含其他对象
- 提供更好的灵活性和可维护性

## 运行示例
执行 `Main.java` 类的 `main` 方法，即可看到完整的面向对象编程示例运行结果。
